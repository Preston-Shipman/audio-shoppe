# wicked-sales-js
E-commerce app

# Technoligies:
- React.js
- Node.js
- Express
- Bootstrap

# Features
- User can view list of all products
- User can view details of all products
- User can add products to their cart
- User can get to the checkout page
- User can enter their purchase information
